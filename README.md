# 📊 Sales Performance Dashboard (Excel)

Interactive sales dashboard built in Excel using Pivot Tables, Pivot Charts, and Slicers.

---

## 🚀 Overview

This dashboard analyzes sales performance across regions and categories, helping identify profitability trends and loss-making products.

---

## 📈 Key Features

- Dynamic filtering with slicers (Region & Category)
- KPI indicators (Sales, Profit, Margin)
- Profit analysis by category
- Regional performance comparison
- Identification of profitable vs loss-making products

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Basic data modeling

---

## 📊 Insights

- Technology is the most profitable category
- Furniture shows lower margins and occasional losses
- Profitability varies significantly by region
